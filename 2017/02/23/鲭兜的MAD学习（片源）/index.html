Chap.1 片源

Question-01：各片源版本及其异同

1.1 TV版

动画主要是三类片源：TV版、DVD版、BD版。
TV版就是指在电视上播出的版本，即是我们平常追番看到的版本。

台标

有很多人做MAD不以TV版为素材的一个原因是台标，台标本身存在感不大，但后期调整容易提高存在感，影响MAD观赏性。

“官方自带弹幕”

左上角的时间对于MAD来说也是碍眼的存在，其他奇奇怪怪的东西也会影响观看。所谓奇奇怪怪的东西，比如BD\DVD的发售日期、（网络上非法利用电视节目的情况非常频繁）法律禁止未经许可在网上传播的警告、棒球比赛延期、洪水地震预警等，最重要的还有保护视力的一段话。
这里简单地介绍一下保护视力的那段话的由来。

光敏感性癫痫症(PSE)

3D龙事件：97年在日本神奇宝贝的第38集电脑战士3D龙在电视上播出后，由于其过闪的画面导致观众产生过敏性反应，即光敏感性癫痫症（PSE：Photosensitive Epilepsy）。各电视台在1998年4月8日共同协议推动电视节目开头警语的标示规章，从这一天开始所有的日本电视节目，都必须加上这类警示语【看电视时，请保持室内明亮，不要太靠近电视机】（针对一般观众）、【观看电视动画的时候，记得保持室内明亮，而且看的时候不要太靠近喔】（针对儿童观众）。详情请参阅日文维基百科“Pokemon Shock”词条。
这里注明：红光和蓝光最为刺激人眼，视神经受到强烈刺激会影响到脑部的控制，轻则不舒服，重则会昏厥或痉挛、全身抽搐。

“提供”画面、“和谐”画面

TV版一般会存在圣骑或者暗牧(现在还出现了织雾僧)，这类一般是有显示赞助商的免费电视台。
免费电视台：有和谐
收费电视台：无和谐，比如ATX

1.2 DVD版

DVD画质 $\le$ BD画质
DVD价格 $<$ BD价格
现在新番一般找BD，DVD一般找老番的时候才用得到。
BD版：画质、内容修正，无圣光暗牧，附送特典，附评论音轨。

1.3 BD版

vol(卷)

BD分多次发售，通常每一个月发售一卷，即Vol。每卷含几集（季番）到十几集（年番）不等。

BD Remux

Blue-ray Disk Remux 将BD以几近无损的方式提取出来的东西。

ISO

isolation 本意是孤立，ISO是镜像文件的一种，类似于压缩包。ISO格式是将碟片中的所有内容打包进这个文件里，它一般是用虚拟光驱才能打开的，也可以直接解压。
ISO格式太大了，所以非特殊情况，不考虑ISO格式。里面的内容一般也是没有经过压制的，基本就是原盘拷贝。

BD-BOX

多了一个“盒子”的概念，指全集，包含所有Vol。老番要出BD，直接出全集，叫BD-BOX。

アニメ

anime（全称animation）本意泛指动画，现在特指日本动画。

clpi

clip-info，储存影片信息的文件，比如影片的编码方式、视频长度、音轨数量等这类文件的属性信息和时间信息，这个是与后面的m2ts（视频文件）相对应的，每一个视频流文件必须对应一个clpi。

mlps

mediaplaylist，储存着播放列表，里面有一个或者多个播放项目（playitem），用来确保你在播放每个文件时能确切播到某个位置。

BDMV

Blue-ray Disk Movie BD光驱专用的格式，资源上标有BDMV的可以理解为蓝光原盘。

index.bdmv

存储原盘目录信息

MovieObject.bdmv

存储播放命令，这里的object是指movie object 影片对象，里面包含了一个可执行的导航命令程序。

xml

xml文件记录的是logo的相关信息，也就是说xml文件一般会和一个图片文件在一起。

id.bdmv

是许可证一类的文件，可以理解为正版证明。
在下载ISO文件时，以上这些大可以不下，可以直接瞄准stream（流，视频流）文件夹。

M2TS

是视频格式的一种，体积极大，与之差不多的还有ts格式，m2ts格式、ts格式常见于BD。这种格式可以被很多非编软件直接导入（非编软件先理解为拿来做MAD的软件）。
总之，一般都下BDrip，犯不着找ISO格式那么大的无损。但是真要是冷番难觅那也没办法，BDMV就BDMV吧。

1.4 补充

U2娘

发布组

U2

一个经常发布BDMV动漫的PT站

PT

Private Tracker，下载依然为BT的原理，但是更为私密，资源仅在站点内部用户之间上传下载。常见的PT站除了U2，还有教育网的PT站六维空间等。

1.5 SP/特典

BD会带有SP，Special（Special Part），中译特典。特典就是附加的内容，可能是图片、声音或是视频。视频可能是见面会、访谈录、演唱会等。还有就是NCOP/NCED，做AMV比较重视。

NCOP~NCED

NCOP/NCED就是没有XX制作XX监督XX声优之类的无字幕的纯净画面。
NCOP：Non-Credit Opening 无字片头
NCED：Non-Credit Ending 无字片尾
此处的“字”指的是人员名单而不是字幕组字幕。

credit/staff

staff是制作人员，那么credit就是参与人员。
去国外网站搜索注意：NCOP的称呼比较常见的有NC、NCOP、Creditless OP、Creditless Opening、ノンクレジット(Non-Credit)。比较少见的有Cleaning Opening、Textless Opening。

ノンテロップ(Non-Telop)

有些视频名字里有ノンテロップ，即Non Telop，Telop是television opaque projector的简称，大意就是为了通过电视摄像机发送文字、图片、照片的装置。换句话说，Telop意指字幕，Non Telop就是没有字幕。

常见命名结尾

有些番的NCOP有好多个，所以有OP1、OP2云云，还有常见命名方式，比如OP ver1、OP ver2、OP verA、OP verB。
同一部番各个OP不一样，或是因为剧情需要，比如影藏人物出场之类的。

ep/episode(集)

还有在OPED命名里要注意的一项是“ep”，全称episode，就是“集”。这一般会在最后一集用到，毕竟很多动画的最后一集都是天国的OPED君，就算ED响起了，但是画面却不是ED原画面，而是动画内容。而这个内容又有credit名单，所以会单独出个视频。
OPED可不一定都是单独的视频，有合在一起的，需要自行把它们拆开。

I帧问题

想把进度条手动拉至1：20
结果进度条自动跳至1：22
即1：22那个画面为I帧（MEPG编码下）
分割同理
若是不经过重新编码而进行视频分割，即所谓的“无损分割”，基本都会自动跳到其他的画面（I帧）来分割而非你选的分割点。在网络上看视频拉进度条拉不到确切你所定位的那一帧而是（跳到）旁边的画面，也是由于I帧问题。

特例

OP1、ED1一般就在第一卷发售，所以搜NCOP的时候可以试试搜索Vol.1。
并不是所有BD都有NCOPED的。
有时在DVD中找到NCOPED，但是BD里面却没有。
一般的番有NC不出，可能是考虑到credit的艺术效果。
与很多OP不同，有可能不是先做画面再做Credit，而是打从一开始就融入了Credit来做的，所以只有一个版本。
BD和DVD两种版本的NCOP还可能不一样。

1.6 特殊版本

重制版

还有一些版本比较特殊，比如重制版。字面理解就是重新制作，全新的，但是却不对。重制应该理解为数码重制或者数码修复。若是重制之后还播出来，那就是数码重映。

数码修复版/数码重映

digital remaster
对母带副本的音视频进行技术处理，将原在胶片上的内容转为数字记录媒体————比如DVD或者BD
remaster字面理解，就是重新制作母带，增强视觉效果，提升画质，修复细节，具体做了什么处理因片而异。
1、提升画质以适应现在的高清电视的播放要求
2、为粉丝提供更好的回顾和收藏
3、拖慢动画进度（所以柯南其实现在有两个目录，你有时发现有些网站的集数对不上，就是因为有个目录列表将数码重映的也算进去了）
母带：Master，概念来源于CD的master tape。因为CD的母带在保存过程中会随着时间慢慢劣化，所以就需要把原有的母带重新录制一遍。母带和CD就好比胶片和数码相片。

导演剪辑版

Director’s Cut，相当于加长版，多了一些内容。一般来说，导演和制作人意见不合时就会产生导演剪辑版，因为制片人偏商业性，导演偏艺术性。

OA版

On The Air的缩写，本义是在广播，这里指电视上播出的版本，即tv版。

Renewal版

简称Renew版，基本就是数码重制版+导演剪辑版的结合。
